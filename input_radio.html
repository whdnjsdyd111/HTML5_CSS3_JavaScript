라디오 버튼 input 속성인 radio는 라디오 버튼을 정의한다. 사용자는 여러 항목 중에서 하나만을 선택할 수 있다. 예를 들어 성별을 입력할 때 라디오 버튼을 사용해 사용자가 남성과 여성 둘 중 하나만 선택하도록 할 수 있다. 이때 type은 radio로 설정해야 하고 name과 value 속성도 반드시 지정해야 한다. 이 때 주의할 점은 name이 같아야 동일한 그룹으로 취급된다는 점이다. 동일한 그룹에서는 하나만 선택할 수 있다. 서버로는 선택된 라디오 버튼의 value 값이 "gender=male"과 같은 형태로 전달된다.
