제출 버튼과 초기화 버튼 input 속성인 submit은 제출 버튼을 정의한다. 제출 버튼은 데이터를 서버로 전송하는 데 사용된다. 데이터는 "name1=value&name2=value2..."의 형태로 action 속성에 지정된 스크립트로 보내진다. action 속성에 지정된 스크립트는 전송받은 입력에 대하여 어떤 처리를 한다.
사용자 아이디 :
사용자가 이이디를 입력하고 "제출" 버튼을 누르면 브라우저는 사용자가 입력한 값을 GET 방식으로 getid.jsp라는 페이지로 보낸다. value 속성을 지정하지 않으면 디폴트로 버튼은 '쿼리 전송'이 된다. input 속성인 reset은 초기화 버튼이다. 이 버튼을 누르면 입력 필드에 입력한 값이 모두 초기화된다. 일반적으로는 입력 양식 안에는 submit 버튼과 reset 버튼이 있어야 한다.
